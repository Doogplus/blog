# 初识 Akka.NET

早就知道 JVM 下的 [Akka][20] 和 .NET 下的 [Akka.NET][2]，但一直没深入了解。

今天花了点时间瞄了一眼 [Akka.NET][2]，有意思，基本把 erlang 的基本概念都实现完毕了，而且社区也不错。

正好找到早期一位作者的 blog，详细记录了 [Akka.NET][2] 的设计历程。有空要全部读一遍，学习下。
