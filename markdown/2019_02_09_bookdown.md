## R Markdown 是啥？

简单说，就是 LaTeX 用烦了，新设计了一种纯文本结构，把 R + Markdown 揉合在了一起，然后写了一推工具，可以导出 PDF、HTML、ePub 等等格式。

 * 《[bookdown: Authoring Books and Technical Documents with R Markdown][5]》
