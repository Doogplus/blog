# 天涯.明月.刀

```
这几天五一长假，在家闲来无事，跑到书城逛逛，在“三联书店”专卖里面又看到了年少时喜爱的武侠小说。。记得以前对之特别喜爱，梦想这也是长大赚钱了，自己买一整套金庸全集来收藏。

现今自己有些米了，少了少时的囊中羞涩，自然忍不住捎上两本小说。。看武侠，最早是从金庸《碧血剑》起，一路“飞雪连天射白鹿，笑书神侠倚碧鸳”。。之后读过梁羽生，自认为梁的文笔与历史，都不如金；也啃过黄易，《大唐双龙传》中群雄逐鹿与武侠气息环环相扣，着实让人着迷，但笔功、人物刻画方面黄还远逊于金；……

古龙的文字，是我最不敢品评的。。最初读古龙，感觉其文字诡异，年轻时的我读起来费力，要花些脑力才能明白他想表达的内容，因而最初对古龙，总是敬而远之。但读过《绝代双娇》，又对其十分欣赏起来，个人认为《绝代双娇》是笔法上最不“古龙”的文字（至少是我读的古龙的作品里面），而更接近金的那种描述性文字。。之后看过楚留香、陆小凤，对古龙编故事的功底，确是五体投地。。

《天涯.明月.刀》以前似乎是看过的，以前没有细细品味过古龙的文字，主要是囫囵吞枣地看情节，如今再次翻开书页，心境却又不同。

~~~~~~~~~~~~~~~~~~~~~~~~~~~~
天涯远不远？

不远！人就在天涯，天涯怎麽会远？

明月是什麽颜色？

是蓝的，就像海一样蓝，一样深，一样忧郁。

明月在哪？

就在他的心里，他的心就是明月。

刀呢？

刀就在他手！

那是柄什麽样的刀？

他的刀如天涯般辽阔寂寞，如明月般皎洁忧郁，有时一刀挥出，又彷佛是空的！

空的？

空空蒙蒙，缥缈虚幻，彷佛根本不存在，又彷佛到处都在。

可是他的刀看来并不快。

不快的刀，为什麽能无敌於天下？

因为他的刀已超越了速度的极限！

他的人呢？

人犹未归，人已断肠。

何处是归程？

归程就在他眼前。

他看不见？

他没有去看。

所以他找不到？

现在虽然找不到，迟早总有一天会找到的！

一定会找到？

一定！
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

文字中透出淡淡的意境，些许“诗”的味道～
```
