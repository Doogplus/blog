安装最新版本的 node.js

 * [https://nodejs.org/en/][2]

创建 react 项目，初始为使用 typescript 的项目。

```
> npx create-react-app myapp --scripts-version=react-scripts-ts

> cd myapp
> npm run
```

自动启动 webserver，并打开浏览器。:-) 这开发傻瓜化程度和当年 Visual Studio 一样了。
